# Agro forte, futuro sustentável

Projeto em HTML, CSS e JavaScript sobre o equilíbrio entre produção e meio ambiente.

## Tecnologias
- HTML
- CSS
- JavaScript

## Objetivo
Mostrar, de forma interativa, como ações sustentáveis ajudam a produzir melhor e preservar recursos naturais.

## Funcionalidades
- Modo escuro
- Placar de ações sustentáveis
- Simulador de uso da água
- Dicas interativas

## Como executar
1. Abra o arquivo `index.html` no navegador.
2. Ou publique no GitHub Pages / Vercel.

## Organização
- `index.html`
- `style.css`
- `script.js`

## Observação
Projeto feito sem frameworks, com arquivos separados.
