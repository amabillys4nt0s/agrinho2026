# Agro Aventura Sustentável

Projeto para o Concurso Agrinho 2026, categoria Programação Front-End.

## Tema

Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente.

## Objetivo

O site funciona como um jogo de plataforma educativo. O jogador atravessa fases, coleta sementes e responde perguntas sobre solo, água e sustentabilidade no campo. A proposta é mostrar que a produção agropecuária pode ser forte quando respeita o meio ambiente e usa tecnologia com responsabilidade.

## Como jogar

- Use as setas do teclado ou os botões da tela.
- Pegue as sementes para ganhar pontos.
- Chegue até a bandeira.
- Responda corretamente à pergunta para liberar a próxima fase.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

O projeto não utiliza frameworks, bibliotecas externas, CSS inline ou JavaScript interno no HTML.

## Organização dos arquivos

- `index.html`: estrutura semântica do site.
- `style.css`: estilos, responsividade, animações e aparência do jogo.
- `script.js`: lógica do jogo, fases, colisões, perguntas, pontuação e interações.

## Recursos de acessibilidade e usabilidade

- Layout responsivo com media queries.
- Botão de modo noite.
- Botão para aumentar fonte.
- Controles por teclado e por botões na tela.
- Textos de retorno nas perguntas.

## Publicação no GitHub

1. Crie um repositório público no GitHub.
2. Envie os arquivos `index.html`, `style.css`, `script.js` e `README.md`.
3. Em Settings > Pages, publique a branch principal.
4. No About do repositório, coloque o link do GitHub Pages.
5. Em Topics, adicione a tag `agrinho`.

## Créditos

Textos, fases, personagem e elementos visuais foram criados para este projeto em HTML, CSS e JavaScript. Recomenda-se que o estudante personalize nomes, perguntas, cores e elementos visuais para reforçar sua autoria antes da submissão oficial.
